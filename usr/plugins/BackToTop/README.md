# BackToTop

- 简单 
- 轻量级

随机三款东方样式,刷新页面改变

## 【食用方法】

Download ZIP, 解压

重命名文件夹为BackToTop

上传至usr/plugins目录

登录后台启用即可

如果主题模板已加载jquery，可以在插件设置内禁用加载jquery
